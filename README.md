# GEOM Lab Website

Responsive website for the Geomorphology of Mountains & Valleys (GEOM) Lab at IISER Mohali.

**Live website:** https://athifsayyaf.github.io/Geom_website/

## Run locally

```bash
pnpm install
pnpm dev
```

Create a production build with `pnpm build`.

Pushes to `main` are deployed automatically through GitHub Pages.

React 19, Vite 6, Phosphor Icons, and locally hosted Manrope / DM Sans fonts.
